## Java Concorrente

- Apresentação de trabalho sobre Java Concorrente, ` Paradigmas de Linguagem de Programação `

----

#### Exercícios do Livro Base:

1. Escreva um programa em Java que realize o cálculo das somas dos
valores de uma matriz 4x4 de números inteiros e imprima o resultado
destas somas na tela. Faça com que o cálculo do somatório de cada
linha seja realizado emparalelo, porthreads;

2. Cinco lebres disputarão uma corrida. Cada lebre pode dar um salto
que varia de 1 a 3 metros de distância. A distância percorrida é de 20
metros. Na corrida, cada lebre dará um salto. Informar quantos
metros ela pulou a cada salto realizado. Em seguida, a lebre pára para
descansar por 2 segundos (sleep). Escreva um programa, utilizando
threads (uma para cada lebre), que informe a lebre vencedora e a
colocação de cada uma delas ao final da corrida. Informar, também,
quantos pulos cada uma delas deu.
